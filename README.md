# Emotion Recognition

## Overview
Facial emotion recognition project using deep learning.
This project compares a CNN model with a MobileNet transfer learning model for facial emotion classification using the FER2013 dataset.

## Dataset
- FER2013 (Kaggle)
- Grayscale facial images
- Emotion categories:
  - Angry
  - Disgust
  - Fear
  - Happy
  - Sad
  - Surprise
  - Neutral

## Models
### CNN
- Custom CNN architecture

### MobileNet (Transfer Learning)
- Pretrained MobileNet
- Fine-tuning for emotion classification

## Results
| Model | Accuracy |
|---------|---------|
| CNN | 56.7% |
| MobileNet | 48.26% |
